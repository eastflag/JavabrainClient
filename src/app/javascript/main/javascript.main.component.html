개요

* 개요

자바스크립트는 3가지 요소로 이루어져 있다.

Core: 자바스크립트 핵심 요소

DOM(Document Object Model) 노드와 노드리스트를 찾고 바꾸고 조작하는 인터페이스 정의

BOM(Browser Object Model): Location, Navigator, Window 관련된 인터페이스 정의

* 학습 목표

표현식과 객체 리터릴 표기법, 생성자함수의 의미

함수 선언식, 기명함수 표현식, 익명함수 표현식, 기명즉시실행함수, 익명즉시실행함수 등 함수 선언 방법

자바와 다른 this의 6가지 다른 의미

변수의 스코프, 호이스팅, 클로저의 의미

프로토타입과 call, apply등의 리플렉션 함수를 이용한 상속 방법
