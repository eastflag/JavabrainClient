* 로드맵 설명
난이도가 쉬운것부터 높은순으로 배열하였다.
자바 문법적으로는 클래스, 다형성, 인터페이스에 대한 문법은 객체지향 개념의 가장 기초가 되는 문법이며 실제 자바 개발시에 가장 빈번히 사용되는 문법은 익명클래스, 제네릭, 컬렉션, 쓰레드 등이다.
자바 중급으로 올라서기 위해서는 디자인 패턴에 대한 공부뿐만 아니라 멀티쓰레드 패턴에 대한 공부가 필요하다.
* 자바 공부 로드맵
제목 자료 분류 난이도 비고
난 정말 JAVA를 공부한 적이 없다구요(책) 출판사 홈페이지 문법 초급 동영상강좌, 소스, ppt교재까지 있음. 다형성에 대한 내용 부족
자바의정석(책) 네이버카페 (필수자바강의 참고) 문법 초급 컬렉션, 제네릭에 대한 설명 부족
소설같은자바(책) 문법 초급 강추. 절판. 중고로 사서 보기를 권함
이것이 자바다(책) 네이버 카페 문법 초급,중급 람다 expression 등 자바 1.8 문법이 많이 소개되어있음
Head First Design Pattern(책) pdf 다운로드 객체지향개념 중급 인터페이스, 다형성등 문법을 완전히 이해한 다음에 봐야할 책
JAVA 언어로 배우는 디자인 패턴 입문 멀티쓰레드 편(책) 객체지향개념 중급 강추. 절판. 자바로 개발한다면 반드시 1번이상 정독해야 할 책. 중급개발자로 올라서기 위해 반드시 익혀야 할 멀티쓰레드 디자인 패턴을 자바 문법 1.5 이전 패턴과 1.5이후 cuncurrent 패키지의 두가지 예제로 설명.
